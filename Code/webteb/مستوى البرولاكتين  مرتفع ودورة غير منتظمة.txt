#  مستوى البرولاكتين مرتفع ودورة غير منتظمة

__شارك __غرد

السؤال

أنا عمري 26 مع دورة شهرية غير منتظمة. في الشهر الماضي بدأت بإجراء فحص تعقب
الجريبٌات المبيضيه واتضح انه لم تحدث الإباضة. أجريت فحص لمستوى الهرمونات في
اليوم الثاني للحيض (الذي حدث بعد أخذ بروفيرا). ماذا تظهر النتائج؟ هل هي
طبيعية؟ TSH 3.29 mIU/L 0.35-4.94 Estradiol 232.0 pmol/L Reference values for
females Follicular phase: 77-921 Mid cycle: 139-2381 Luteal phase: 77-1145
Postmenop.(not on HRT): 36.7-102 Postmenop.(on HRT):36.7-161 FSH 6.2 mIU/mL
Reference values for females Prepubertal: LH 7.2 mIU/mL Reference values for
females Follicular phase:2.39-6.60 Mid cycle: 9.06-74.24 Luteal phase:
0.9-9.33 Postmenopausal: 10.39-64.57 Prolactin 671.9 mIU/L 25-628

الجواب

معظم النتائج مرضية، باستثناء اختبار البرولاكتين وال- LH. نتائج هاذين الفحصين
تشذ قليلا عن المجال الطبيعي. قيم البرولاكتين العالية يمكنها بالتأكيد أن
تفسرعدم انتظام الدورة الشهرية عندك.  
  
يجب عليك العودة إلى الطبيب الذي أحالك إلى تنفيذ هذه الاختبارات، ومواصلة
التحقيق. على أي حال، فانه لا يوجد سبب للقلق، لأنه يمكن موازنة القيم الغير
طبيعية بواسطة الأدوية، من المهم فقط أن نفهم مصدر المشكلة.
